Rosbridge demo

How to connect:

  • Start rosbridge server by entering ros2 launch rosbridge_server rosbridge_websocket_launch.xml inside terminal
  • Server should run at port 9090 by default

    • To publish string messages:

    • Push messages to webpage by opening new terminal and entering messages in the format: ros2 topic pub -1 /my_topic std_msgs/String "data: 'Hello World'"

    To publish static images (.jpg format):

    • Build web_publishers package by running colcon build --packages-select web_publishers
    • Source the setup files by running source install/setup.bash
    • Publish sample.jpg (or any image as long as you replace IMG_PATH in imgpublisher.py) by running ros2 run web_publishers image_publisher

    To publish controller data:

    • Make sure your Ubuntu device detects controller, either via USB or Bluetooth
    • Start a joy node by entering ros2 run joy joy_node inside terminal
